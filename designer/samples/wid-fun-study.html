widget对象

基于Canvas的基础控件对象

方法说明

名称 说明
Background

类型： function(a)，其中a为设置的背景值。例子： 


var a = {
	filltype: "color",
	color: "yellow"
}
widgetsName.Background(a);

作用： 将自定义的背景对象设置为某个widgets的背景值
Scale

类型： function(a)，其中a为控件伸缩值。例子： 


var a = 2;
widgetsName.Scale(a);

作用： 给某个widgets对象设置伸缩比例
Width

类型： function(a)，a为指定的width值。例子： 


var a = 100;
widgetsName.Width(a);

作用： 给控件设置宽度，默认单位为PX
Height

类型： function(a)，a为指定的Height值 


var a = 100;
widgetsName.Height(a);

作用： 给控件设置高度，默认单位为PX
fontSize

类型： function()

作用： 获取控件字体大小
clean

类型： function()

作用： 清空控件本身
persist

类型： function()

作用： 序列化要保存的控件对象
paintTo

类型： function(canvas, scaleX, scaleY)，canvas为某个canvas对象名，scaleX为该控件的水平伸缩比例，scaleY为 该控件的垂直伸缩比例

作用： 将指定控件按照一定的伸缩比例在画布canvas上画出来
paint

类型： function()

作用： 将控件本身画出来
Widgets

类型： function(a)，a为某个控件的name值、序号或者对象本身

作用： 根据a的值去获取到指定的子控件对象
Presenters

类型： function(a)，a为某个presenter对象的name值或者序号

作用： 根据a的值去获取到控件所在的presenter对象
downDepth

类型： function()

作用： 将当前控件置后一级
downStep

类型： function()

作用： 将当前控件置前一级
topDepth

类型： function()

作用： 将当前控件置底
topStep

类型： function()

作用： 将当前控件置顶
appendPresenter

类型： function(presenter)，presenter为某个画布对象

作用： 在当前控件上添加画布对象
appendWidget

类型： function(widget)，widget为某个控件对象

作用： 在当前控件上添加控件对象
restoreChildren

类型： function(_widgets,opt)，_widgets为序列化后的控件对象，opt为需要更新和添加的一些属性值

作用： 恢复_widgets对象的所有子控件
removeWidget

类型： function(widget)，widget为某个控件对象

作用： 在当前控件上移除控件widget
RotatePoint

类型： function(x,y)，x、y分别为x轴、y轴坐标值

作用： 设置控件的旋转点的位置
getImageData

类型： function(x, y, width, height,targetwidth,targetheight)

作用： 获取当前控件指定位置和大小的图像，并根据tagetwidth和targetheight的值计算出的比例压缩成指定的大小，保存到“image/png”
getImageBlob

类型： function(x, y, width, height,type)

作用： 获取当前控件指定位置和大小的图像,按照指定的图像type值，压缩并转化成一串图像编码

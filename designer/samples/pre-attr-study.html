presenter对象

展现和操作控制widget的画布对象

属性说明

名称 说明
name

类型： String

作用： 设置画布唯一的name值，默认值为"presenter"
title

类型： String

作用： 设置画布的title值，默认值为"presenter"
dom

默认值： $("body")

作用： 设置画布渲染在哪个dom节点下，一般设置为ID值
type

类型： String

作用： 设置画布的类型，默认为"panel"，不更改
width

类型： int

作用： 设置画布的宽度，默认值1024PX
height

类型： int

作用： 设置画布的高度，默认值800PX
showgrid

类型： boolean

作用： 设置是否显示网格线，默认值为"false"
dpi

类型： int

作用： 设置显示的每个网格的边长，默认值为8，单位是PX，可不写单位
scale

类型： int

作用： 设置画布伸缩比例，默认值为1，不伸缩
autoalign

类型： boolean

作用： 设置控件移动时，是否按网格线自动对齐，默认值为"true"
background

作用： 设置画布背景，具体属性和用法与widgets对象一致

默认值： 


background: {
	filltype: "color",
	color: "#ABABAB"
},

gradient

作用： 当背景的filltype为渐变色gradient时，设置gradient的属性。具体用法与widgets对象一致。

默认值： 


gradient={
    type:"liner",
    begincolor:"white",
    endcolor:"gray",
    angle:90,
    radius:45
},

border

作用： 设置画布的边框样式，具体属性和用法与widgets对象一致。

默认值： 


border: {
    width: 1,
    type: "solid",
    color: "none"
},

注释文本框note

用法： $.widgets("note", {})

描述：定义一个名为"note"的widgets对象，在画布上绘制一个注释文本框note。 它继承了“actionWidget”的属性和方法，增加了段落属性paragraph；根据字体和段落属性值来计算大小，使文字换行显示的方法calculateSize； 重写drawTextTo方法，将文字按照设置的属性和大小绘制在画布上。下面是改写的一些属性和方法说明。

属性说明:

名称 说明
name

文本框的name值，默认为“name”
text

文本框内部显示的文字，默认为“note”
width

note的宽度，默认为160，单位是PX
height

note的高度，默认为140，单位是PX
corner

设置为带圆角的note，默认值： 

corner: {
	type: "round",
	radius: 16
},

background

给note添加背景，默认为黄色： 

background:{
    filltype: "color",
    image: "",
    imageType: "fill",
    color: "yellow"
},

font

给note添加默认的字体样式

默认值： 

font:{
    style:"normal",
    weight:"normal",
    family:"Arial",
    size:"10pt",
    color:"blue",
    fill:true
},

paragraph

note是注释文本框，新增一个段落的属性，用于改变文字间的样式。包括下列几个参数：
属性名 类型 说明
linespace int 设置行间距
textalign String 设置文字水平对齐方式，有三种，分别为：left-左对齐,center-中对齐,right-右对齐
textvalign String 设置文字垂直对齐方式，有三种：top-上对齐,middle-中对齐,bottom-下对齐

默认值： 


paragraph: {
	linespace: 10,
	textalign: "center",
	textvalign: "middle"
},

autosize

类型： boolean

说明： 大小自适应，默认为false

方法说明：

名称 说明
persist 序列化要保存的"note"对象,并添加了属性paragraph的值 
				
this.persist = function() {
    var r = widget.persistproperty(this);
    if(this.paragraph) {
        r.paragraph = {};
        $.extend(r.paragraph, this.paragraph);
    }
    return r;
};
calculateSize 根据字体和段落属性值来计算大小，使文字能够按照规则换行显示
drawTextTo 重写该方法，将文字按照设置的属性绘制在note边框内。若文字高度超过note本身高度，note的高度会随着文字增加。

widget对象

基于Canvas的基础控件对象

属性说明

名称 说明
name

类型： String

作用： 设置控件唯一的name值，默认值为"widget"
type

类型： String

作用： 设置控件的类型，默认为“widgets”，不更改
text

类型： String

作用： 设置标注控件的“text”值，在控件内部显示
x

类型： int

作用： 设置控件在画布上显示的x轴的坐标值
y

类型： int

作用： 设置控件在画布上显示的y轴的坐标值
width

类型： int

作用： 设置控件的宽度,默认值为275,单位为PX
height

类型： int

作用： 设置控件的高度,默认值为140,单位为PX
minwidth

类型： int

作用： 设置控件的最小宽度，默认值为36，单位为PX
minheight

类型： int

作用： 设置控件的最小高度，默认值为36，单位为PX
alpha

类型： float

作用： 设置控件的透明度 ，范围为0~1 ，可以是小数，如0.2,默认值为1，不透明
editable

类型： boolean

作用： 设置控件是否可编辑，默认为false
visible

类型： boolean

作用： 设置控件是否在画布上可见，默认为true
scaleX

类型： int

作用： 设置控件的水平伸缩比例,默认值为1
scaleY

类型： int

作用： 设置控件的垂直伸缩比例,默认值为1
scale

类型： int

作用： 设置控件的伸缩比例,默认值为1
margin

类型： int

作用： 设置控件的外边距，默认值为5，单位是PX
rotate

类型： int

作用： 设置控件的旋转角度，默认值为0
rotatepoint

作用： 设置旋转点的位置和偏移的距离,其中x,y为坐标位置值，offsetx和offsety分别设置控件在水平和垂直方向偏移的距离

默认值： 


rotatepoint: {
    x: 0,
    y: 0,
    offsetx: 0,
    offsety: 0
},

background

作用： 设置控件背景，其中"filltype"为填充方式，共有4种，分别为：
1、none--无填充；
2、color--颜色填充；
3、gradient--渐变色填充；
4、image--图片填充 。
默认值为：

					
background: {
	filltype:"color",
	color : "#ABABAB"
},

其他用法：

1、gradient-渐变色填充,结合另一个属性“gradient”来设置，详见下一个属性“gradient” 

					
background: {
	filltype: "gradient"
},
gradient: {
	type: "liner",
	begincolor: "white",
	endcolor: "gray",
	angle: 90
},

2、image-图片填充，其中，“image”为图片路径，“imageType”为图片填充方式，共4种，分别为：
（1）repeat-按照原比例重复图片；
（2）center-显示图片中心部分；
（3）fit-自适应画布大小，保留边框；
（4）fill-图片按比例填满画布。

					
background: {
	filltype: "image",
	image: "xxx.jpg",
	imageType: "fill"
}

默认值： 


background: {
    filltype:"color",
    color : "none",
    image : null,
    imageType : "center"
},

gradient

作用：当背景的filltype为渐变色gradient时，设置gradient的属性。 渐变的方式有3种，分别为:
1、none-无;
2、liner-线性；
3、radial-径向；
可根据需求设置对应的开始颜色“begincolor”和结束颜色“endcolor”；“angle”为线性渐变的角度， 只对线性渐变方式有作用；“radius”为径向渐变中中心圆的半径大小，只对径向渐变方式有作用。

用法(默认值)： 

					
gradient: {
	type: "liner",
	begincolor: "white",
	endcolor: "gray",
	angle: 90,
	radius: 45
},

border

作用：设置画布的边框，共有三个属性，其中“width”用于设置边框宽度， 单位默认为PX；“color”用于设置边框的颜色，“type”用于设置边框的样式，共有3种，分别为：
1、solid--实线；
2、dotted--点状线；
3、dashed--虚线。

用法： 

					
border: {
	width: 1,
	type: "solid",
	color: "red"
},

corner

作用： 设置控件边框的形状，共有两种，分别为：
1、round--圆角；
2、rect--方角。
其中，“radius”属性为设置圆角的半径，当type为round时才起作用。

默认值： 


corner: {
    type:"rect",
    radius:16
},

shadow

作用： 给控件添加阴影效果，有下列几个属性：

属性名 类型 说明
color String 设置阴影颜色
offsetX int 设置阴影水平偏移的距离，正值即为向右偏移，负值则相反
offsetY int 设置阴影垂直偏移的距离，正值即向下偏移，负值则相反
blur int 设置阴影的模糊程度，其值不能为负值，值越大，阴影越模糊，反之越清晰，不需要阴影模糊可以将其值设为0

默认值： 


shadow: {
    color : "none",
    offsetX : 4,
    offsetY : 4,
    blur : 4
},

font

作用： 给控件设置显示的字体样式，有下列属性：

属性名 类型 说明
style String 设置字体样式，分为“normal”正常和“italic”斜体
weight String 设置字体粗细程度，分为“normal”正常和“bold”粗体
family String 设置显示的字体名
size String 设置字体大小,单位为PT
color String 设置字体颜色
fill boolean 设置文字是空心还是实心，默认为true,实心

默认值： 


font: {
    style: "normal",
    weight: "normal",
    family: "Arial",
    size: "10pt",
    color: "black",
    fill: true
},

actionWidget

描述：继承widgets对象，添加了阴影效果、超链接属性和实现了可以点击超链接跳转的事件，作中间件，不可直接使用

属性说明:

名称 说明
hyperlink

作用：添加超链接，具体属性如下：
属性名 说明
type 超链接类型，有三种，分别为“url”、“window”和“command”
url 若type值为“url”，则需要添加该属性，填写跳转的url值
target 网页打开形式
1、“_blank”--在新窗口中打开被链接文档；
2、“_self”--在相同的框架中打开被链接文档；
3、“_parent”--在父框架集中打开被链接文档；
4、“_top”--在整个窗口中打开被链接文档；
5、“framename”--在指定的框架中打开被链接文档。

默认值： 


hyperlink: {
	window: "",
	url: "",
	command: "",
	type: "url",
	target: "_blank",
},

shadow

给该控件添加黑色的阴影效果

默认值： 


shadow: {
    color : "black",
    offsetX : 4,
    offsetY : 4,
    blur : 4
},

事件说明：

名称 说明
click

根据hyperlink的属性值，给控件添加一个可以点击超链接跳转的事件。其中，当editable的属性值为“false”时，点击事件才生效。
